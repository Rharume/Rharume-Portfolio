Este é o meu portfólio pessoal online desenvolvido para apresentar quem eu sou, meus conhecimentos em front-end e meus projetos. O site está hospedado em:  
https://rharume-portf-lio.vercel.app/

## Descrição

O portfólio é um site estático criado com HTML, CSS e JavaScript. Nele você encontra uma apresentação pessoal, uma seção sobre mim, meus projetos e informações de contato. O objetivo é mostrar meu progresso no desenvolvimento web como estudante de Sistemas de Informação e desenvolvedora front-end.
